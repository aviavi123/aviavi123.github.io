Echo Pitch

Mac / iOS Plugin

Pitch-shifting multi-delay and harmonizer for cascading delays and loops.

Echo Pitch is an Audio Unit (AUv3) plugin that combines a flexible multi-tap delay with real-time pitch shifting. Each repeat can be transposed, so echoes cascade up or down in pitch — perfect for shimmering ambient textures, harmonized loops, and otherworldly delay trails inside your DAW or favorite AUv3 host on Mac and iOS.

Key Features

  • Pitch-shifted feedback: Transpose each delay repeat to build cascading, evolving pitch trails.
  • Multi-delay engine: Layer multiple delay voices for rhythmic and ambient patterns.
  • Harmonizer mode: Add tuned intervals on top of your signal for instant harmonies.
  • AUv3 everywhere: Runs as a plugin on both macOS and iOS hosts.
